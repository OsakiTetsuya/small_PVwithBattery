# JEPX売上シミュレーター 使用ガイド

## セットアップ

### 1. 環境構築

```bash
# リポジトリのクローン
cd /path/to/small_PVwithBattery

# 依存パッケージのインストール
pip install -r requirements.txt
```

### 2. アプリの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、アプリが表示されます（通常は http://localhost:8501）。

## 使い方

### ステップ1: データ取得

1. 左サイドバーから「データ取得」を選択
2. 「市場価格データ」タブで、JEPX市場価格のCSVファイルをアップロード
   - フォーマット: `datetime,area,area_price_yen_per_kwh`
3. 「データベースに保存」ボタンをクリック

### ステップ2: 発電所登録

1. 左サイドバーから「発電所登録」を選択
2. 発電所情報を入力:
   - 発電所名
   - 設置エリア
   - 太陽光容量 [kW]
   - PCS容量 [kW]
   - 系統売電上限 [kW]
3. 1時間発電量データのCSVファイルをアップロード
   - フォーマット: `datetime,generation_kwh`
4. 「発電所を登録」ボタンをクリック

### ステップ3: シミュレーション実行

1. 左サイドバーから「シミュレーション実行」を選択
2. シミュレーション設定を入力:
   - シミュレーション名
   - 発電所選択
   - 対象エリア
   - 対象期間
3. 蓄電池設定（蓄電池を使用する場合）:
   - 蓄電池容量 [kWh]
   - 最大充放電出力 [kW]
   - 充放電効率
   - SOC範囲
   - 放電方式（上位価格コマ方式 / 価格しきい値方式）
4. 「シミュレーション実行」ボタンをクリック

### ステップ4: 結果分析

1. 左サイドバーから「結果分析」を選択
2. 各種グラフで結果を確認:
   - 発電量と価格
   - 蓄電池挙動
   - 売上分析
3. 「データ」タブからCSVファイルをダウンロード可能

## 注意事項

### MVP版の制限事項

- JEPX公式サイトからの自動データ取得は未実装（CSV手動アップロードで代替）
- 各電力会社サイトからの自動データ取得は未実装（CSV手動アップロードで代替）
- エリア太陽光実績による補正は簡易実装
- 託送料、手数料、インバランス、消費税は考慮していません
- 本アプリの計算結果は参考値であり、実際の売上を保証するものではありません

## トラブルシューティング

### エラー: "データが見つかりません"

- 「データ取得」ページで対象期間とエリアの市場価格データが登録されているか確認してください
- 「発電所登録」ページで発電所が登録されているか確認してください

### エラー: "datetime列のパースに失敗"

- CSVファイルのdatetime列が `YYYY-MM-DD HH:MM` フォーマットになっているか確認してください
- 例: `2025-07-01 10:00`

### 蓄電池の動作が期待と異なる

- 蓄電池の容量、最大充放電出力、SOC範囲の設定を確認してください
- 放電方式（上位価格コマ / しきい値）の設定を確認してください
- 初期SOCが適切に設定されているか確認してください
